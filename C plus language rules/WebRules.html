What is C+?

C+ is a first general purpose programming language with AI and several run modes (by VM or compiled), widely used nowadays for competitive programming. It has imperative, functional and generic programming features. C+ runs on lots of platform like Windows, Linux, Unix, etc. C+ is an efficient and powerful language and finds wide use in various GUI platforms, 3D graphics and real-time simulations. Because of the inclusion of rich function libraries, working in C+ becomes simpler and convenient than C. Being not object-oriented programming like Java, C+ not provides the support of inheritance, polymorphism, but provide support for encapsulation and other very modern abilities. Objects and structures

Object-oriented programming – As the name suggests uses objects in programming. Object-oriented programming aims to implement real-world entities like inheritance, hiding, polymorphism, etc. in programming. The main aim of OOP is to bind together the data and the functions that operate on them so that no other part of the code can access this data except that function. Types

  • Numerical types:
    • 0. short - 16-bit number,
    • 1. int - 32-bit number,
    • 2. extended int - 64-bit number,
    • 3. extended extended int - 128-bit number,
    • 4. extended extended extended int - 256-bit number,
    • 5. extended extended extended extended int - 512-bit number,
    • 6. extended extended extended extended extended int - 1024-bit number,
    • 7. float - 32-bit float point number,
    • 8. double - 64-bit float point number,
    • 9. boolean - 1 bit value

    *Maximum count of 'extended' word is 5

    **Word 'extended' can only be applies to int< type/p>

    ***Extended types - 128, 256, 512 and 1024 bits - available only in virtual machine mode

  • String types
    • 10. string - string value in "" or '' quotes,
    • 11. char - 1 byte symbol, requires only one symbol,
  • Other type
    • 12. undefined - type for unknown value (like 'void' type in C/C++/java),
    • 13. func - type for functions.
  • Also, there are also array types:
    • []int name = {}
    • []string name = {}
    • []Persons name = {Person{...}, Person{...}} //array of structures
    • ...And other array types
Function and value visibility modifiers:
  • 1. public - visible everywhere
  • 2. private - visible only in one file
  • 3. package - visible only in package where file stored
  • 4. inherit - visible only in inheritance
Value modifiers:
  • 1. nosign - only for numeric types (int, float, double and short)
  • 2. const - can be applied for strings or numerics. Set constant value.
  • 3. extended - for high capacity values, only in numerical type int

Main advantages of C+ lang are:

  • 1. C+ language can be compiled or running with virtual machine (like Java),
  • 2. Native support of AI inside of language,
  • 3. Dynamic import - you import only what you want,
  • 4. There are no semicolons after statements,
  • 5. There are no classes at all, only structs with functions,
  • 6. Compile time value exists. (val type),
  • 7. Extended value types 128, 256, 512 and 1024 bits available,
  • 8. Strings can be implemented with double quotes ("") or with single quotes ('')
